*@ @*Collections are 'buckets' of group documents (aka objects). Documents are objects defined in JSON format. For each defined collection the number of records is shown. You can delete a collection and all the objects belonging to it by clicking on the 'Delete...' button.*@ @*
*@ @*To create a new collection, click on the 'New Collection' button.*@

文档集合是一组文档的存放的地方。文档是以json格式定义的对象。对应每一个集合都有一个数值显示该集合中保存的记录（文档）的数量，通过选择每一项最后的删除按钮，可以删除一个文档集合，该操作同时也会删除集合中包含的所有文档。
可以通过点击“新建”按钮来创建一个新的文档集合。
